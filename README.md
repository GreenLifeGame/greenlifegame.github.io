# GREEN LIFE

The first fixed-supply play-to-earn game on Solana. Grow, breed, and earn **$GREEN** — 420 billion tokens minted once, never again.

**Status:** In active development. Targeting launch in ~3–6 months. No token exists yet and nothing is for sale.

This repository hosts the public info site (GitHub Pages):
- `index.html` — overview, tokenomics, features, roadmap
- `plants-and-boosts.html` — the full strain & boost codex (art + stats)
